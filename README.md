# 深圳国际科技信息中心门户

深圳国际科技信息中心门户前端原型，首页负责业务导航，四个核心业务使用独立落地页承载详细介绍与系统入口。

- 五屏首屏轮播与近期运营活动广告位
- AI 科研、战略咨询、未来教育、科技评价业务全景
- 四个核心业务独立落地页
- 战略咨询页内嵌科技信息展示系统
- 多级导航、工作台与管理后台快捷菜单
- 站内搜索、热词与搜索历史
- 桌面端和移动端响应式布局

## 本地预览

直接使用浏览器打开 `index.html`，或通过本地静态服务器预览：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000/`。

## 页面结构

- `index.html`：门户首页
- `ai-research.html`：AI 科研
- `strategy.html`：战略咨询
- `education.html`：未来教育
- `evaluation.html`：科技评价
- `landing.css`、`landing.js`：落地页共享样式与交互
