# 深圳国际科技信息中心门户

以前一版 GitHub Pages 首页为视觉与交互基线，重新组织四大核心业务：

- 多级导航与快捷菜单
- AI 科研、战略咨询、未来教育、科技评价四业务首屏轮播
- 首屏下方可信资源与个性化推荐，支持分类与换一批
- 四个业务独立落地页
- 首屏下方四段大幅核心业务展示，每个板块保留一个直接体验入口
- 战略咨询科技信息展示系统页内嵌入
- 合作伙伴与生态模块
- 综合资源搜索落地页，包含模式、筛选、分类、排序与分页
- 个人中心与消息中心（默认 3 条未读消息）
- 桌面端和移动端响应式布局

## 本地预览

直接使用浏览器打开 `index.html`，或在当前目录启动静态服务器：

```bash
python3 -m http.server 8000
```

## 页面结构

- `index.html`：门户首页
- `ai-research.html`：AI 科研落地页
- `strategy.html`：战略咨询落地页
- `education.html`：未来教育落地页
- `evaluation.html`：科技评价落地页
- `home-additions.css`：首页新增业务与合作生态样式
- `portal-extras.css`：推荐、搜索、个人中心与消息中心样式
- `landing.css`、`landing.js`：四个落地页共享样式与交互

## 在线发布

仓库使用 GitHub Pages 从默认分支根目录发布。
